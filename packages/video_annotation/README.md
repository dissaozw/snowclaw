# SnowClaw Video Annotation

Draws skeleton overlay, COM plumb line, and biomechanical metrics onto video frames.

## Features

- **Skeleton drawing** — Joint dots colored by confidence (green/yellow/red), bone lines
- **COM plumb line** — Vertical balance reference from center of mass
- **Metrics overlay** — Knee flex angle, inclination, COM height %
- **Video encoding** — Annotated output MP4 with H.264, preserves original audio
